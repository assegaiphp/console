# Assegai Console 0.9.4 Release Notes

Released: 2026-06-07

## Overview

Assegai Console 0.9.4 keeps the CLI help output focused on application-development workflows. The public command list has been cleaned up so only supported user-facing commands are shown.

## Highlights

### Cleaner command help

- Removed the unsupported `updates:scaffold` entry from `assegai list`.
- Kept normal project commands unchanged.
- Added regression coverage so the public command list stays focused on supported application workflows.

## Upgrade Notes

- No action is required for existing projects.
- No project files, configuration values, or migration files need to be changed.
- Supported project workflows such as `new`, `serve`, `generate`, `database`, `migration`, `update`, and `wc` commands are unaffected.

## Validation

This release verifies that `updates:scaffold` is absent from the public CLI command list.
